Welcome to Just Another Sentiment Analyzer

This is a search engine that can help you to check which is the sentiment about topics.

You can simply search a topic writing it in the search bar, or you can add tags writing down keywords followed by a comma (example: Tesla, Bitcoin, The Witcher 3).

You can search between "recent" and "popular" news by clicking on the rainbow word at the top of the bar.

In the end, you can choose some options by selecting the icons on the bottom of the search bar: the first option allows you to ignore neutral values showing only "Positive" and "Negative" sentiments, the second one combining all keywords selected in a single query.

Notice that you can use # (hashtag) followed by a word to include a tag (example: #HODL) or the @ (at) to include a username (example: @elonmusk)
